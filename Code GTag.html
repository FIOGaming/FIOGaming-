Gorilla Tag

Prix : 0€

Type : Mini jeu

Plateforme : VR

Date de sortie : 12 février 2021

Éditeur : Another Axiom Inc.

Gorilla Tag est un "jeu de singe". Cela signifie que vous n'avez pas de jambes et devez vous déplacer avec de grands mouvements de bras. Le mode Tag est le mode de base. C'est un peu comme jouer à chat mais en VR. Il existe de nombreux modes de jeux en ligne, ainsi qu'une boutique de cosmétiques, et de (très) nombreuses maps, comme des maps horreur, exploration etc... J'ai essayé une de ces maps avec Le_Poulet_VR (lien ci-dessous).
