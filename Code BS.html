Brawl Stars est un jeu de combat avec une vingtaine de mode de jeu. Il y a une centaine de personnage jouables, avec différents pouvoirs rendant le gameplay en jeu infini. Vous pouvez aussi jouer en classé pour plus de compétition. Brawl Stars offre des évènements réguliers ce qui rend le jeu attractif. Vous pouvez vous entrainer avec des bots, et jouer avec vos amis.
