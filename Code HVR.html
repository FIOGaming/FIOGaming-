Hitman 3 VR

Prix : 27,99€

Type : Action, Aventure

Plateforme : VR

Date de sortie : 20 janvier 2022

Éditeur : IO Interactive A/S

Hitman 3 VR est la réadaptation du jeu Hitman 3 mais en VR. Vous devez infiltrer des lieux sécurisés pour tenter d'éliminer une ou plusieurs cibles, mais sans vous faire repérer. Ce jeu offre un très bon gameplay, et offre de (très) nombreuses possibilités d'éliminer une cible. Vous pouvez récuperer le personnages d'autres personnes, pour accéder ç des lieux interdits d'accès.
